# stigma
Search replace tool for wordpress database.(sqlite, ...)

# Usage

```
          __  .__
  _______/  |_|__| ____   _____ _____
 /  ___/\   __\  |/ ___\ /     \\__  \
 \___ \  |  | |  / /_/  >  Y Y  \/ __ \_
/____  > |__| |__\___  /|__|_|  (____  /
     \/         /_____/       \/     \/
```

Usage:
  php stigma <database_file_path> <target> <replace>

Example:
  php stigma ./ht.sqlite localhost 160.122.111.11
